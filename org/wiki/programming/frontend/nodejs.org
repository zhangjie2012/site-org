* 安装 14.x

** Ubuntu

#+begin_src sh
  curl -sL https://deb.nodesource.com/setup_14.x | sudo bash -
  sudo apt install nodejs
#+end_src

如果 update 时报错 =NO_PUBLIC= 在 sources.list 中对应的 deb 后面新增 ~[trusted=yes]~  [fn:1]。
